# follow

仅作为学习使用的一款聊天交友APP。  
> 项目介绍  


前端使用Flutter构建，后端使用netcore，（因本人后端技术实在太垃圾，后端选择不开源QAQ），使用websocket交互，未使用三方IM，前台缓使`sqllite`缓存聊天信息。  
本项目刚开始启动，作为一个从0开始的项目，我还有很多设想，会在职业生涯中慢慢完善。
